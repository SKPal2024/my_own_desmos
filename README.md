📈 Dynamic Graph Plotter (Mini Desmos)

A web-based graph plotting application built with Flask and Matplotlib that allows users to visualize mathematical equations dynamically. Users can input an equation involving the variable x, specify the plotting range, and instantly generate the corresponding graph.

🚀 Features
📊 Plot mathematical equations dynamically.
🧮 Supports common mathematical functions (sin, cos, tan, log, sqrt, exp, etc.).
🔢 Customizable range for the x-axis.
🖼️ Graphs rendered as images using Matplotlib.
🌐 Simple and intuitive web interface built with Flask.
🔒 Safer evaluation of user-input equations.
📦 Lightweight and easy to deploy.

| Technology     | Purpose                                                     |
| -------------- | ----------------------------------------------------------- |
| **Flask**      | Web framework for handling requests and rendering templates |
| **Matplotlib** | Plotting graphs                                             |
| **NumPy**      | Numerical computations                                      |
| **HTML/CSS**   | Frontend user interface                                     |
| **Python**     | Backend logic                                               |
